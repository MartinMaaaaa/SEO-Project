# Public Prompt Templates

These prompts are safe to upload because they do not contain real analytics data, API credentials, private URLs, or project-specific conclusions.

Rules:

- Keep prompts generic.
- Tell AI to read local data sources instead of embedding metrics in the prompt.
- Do not add screenshots, raw exports, token values, or concrete analytics results.
- If a prompt starts to mention real data, move it to a private local folder outside Git.

Available templates:

- `gsc_opportunity_analysis.md`
- `ga4_engagement_analysis.md`
- `pagespeed_technical_triage.md`
- `monthly_seo_report.md`
